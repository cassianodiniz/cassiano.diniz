# cassiano.diniz — marketplace de plugins

Marketplace de plugins de **Claude Code** do Cassiano Diniz. Ferramentas de desenvolvimento e
automação — instale com um comando.

## Como usar

```
/plugin marketplace add cassianodiniz/cassiano.diniz
/plugin install <plugin>@cassiano.diniz
```

## Plugins disponíveis

| Plugin | O que faz |
|---|---|
| `claudex-dev` | Template de plugin de desenvolvimento com revisão integrada do Codex — ciclo completo: investigação → plano → implementação → revisão. |
